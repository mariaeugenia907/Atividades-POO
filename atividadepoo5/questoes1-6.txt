1- A Classe é a representação da estrutura do objeto, alem de representar uma abstração de objeto.
Sendo que a classe define elementos de dados que um objeto
contém (atributos); o comportamento que o objeto possui (métodos) 
e a maneira como esses elementos de dados e métodos podem ser acessados (interface). 
Assim, os objetos possuem características próprias,denotadas por atributos; 
possuem funções (comportamentos – específicos), dados por métodos e podem ser categorizados,
agrupados, e uma classe descreverá todos os objetos de um tipo particular.
Portanto, classe é definição do tipo e representa um conjunto de objetos de mesmo tipo
e objeto é cada instância derivada da classe; é um elemento do conjunto representado pela
classe.
Exemplo:
classe Carro {
   modelo: String
   marca: String
   cor: String
   combustivel: String
   
}
Objeto do tipo carro
   Modelo: Fiesta
   marca: Ford
   cor: branco
   combustivel: gasolina
  
2-Atributos de uma classe ou propriedades são o que descreve um intervalo de valores que as instâncias da classe podem apresentar.
Um atributo é uma variável que pertence a um objeto e os dados de um objeto são armazenados nos seus atributos, ou seja,
são informações sobre o objeto, dados que podem ser armazenados.
Alem disso, podem ser coletivos ou individuais.
Enquanto que, Os métodos são procedimentos ou funções que realizam as ações próprias do objeto. 
Assim, os métodos são as ações que o objeto pode realizar. 
Tudo o que o objeto faz é através de seus métodos, pois é através dos seus métodos que um objeto se manifesta, 
através deles que o objeto interage com os outros objetos.
Exemplo
Classe caes
atributos: nome, peso, cor dos olhos, quantidade de patas
metodos: falar, andar, comer, dormir
